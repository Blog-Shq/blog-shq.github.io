「HDU3507」Print Article - 斜率优化DP

发表于 2019-08-08 | 属于 OI | 阅读次数：

Link: HDU3507

题面

题面大意

对于长度为 $n$ 的序列 ${a}$ , 将其分为几段, 对于段 $[l,r]$ 的花费为 $\left(\sum_{i=l}^r a_i\right)^2+M$, 求最小花费

Problem Description

Zero has an old printer that doesn’t work well sometimes. As it is antique, he still like to use it to print articles. But it is too old to work for a long time and it will certainly wear and tear, so Zero use a cost to evaluate this degree.
One day Zero want to print an article which has N words, and each word i has a cost Ci to be printed. Also, Zero know that print k words in one line will cost

$$\left( \sum _ { i = 1 } ^ { k } C _ { i } \right) ^ { 2 } + M $$

M is a const number.
Now Zero want to know the minimum cost in order to arrange the article perfectly.

Input

There are many test cases. For each test case, There are two numbers N and M in the first line (0 ≤ n ≤ 500000, 0 ≤ M ≤ 1000). Then, there are N numbers in the next 2 to N + 1 lines. Input are terminated by EOF.

Output

A single number, meaning the mininum cost to print the article.

题解

首先写出特征方程:

$f_i$ 表示 $[1,i]$ 的最小花费, $sum_i$ 表示前缀和, 那么有

$$f_i = \min _ { 1 \leq k < i } \left{ f_k + ( \operatorname { sum }_i - \operatorname { sum }_k ) ^ { 2 } + M\right}$$

直接算就是 $O(n^2)$ 的, 考虑进行优化

对于 $dp_i$ 考虑当在 $j$ 处转移比 $k$ 处更优时, 即

$$\begin{align}dp_j+(sum_i-sum_j)^2 + M &< dp_k+(sum_i-sum_k)^2 + M\dp_j+sum_j^2-2\times sum_i\times sum_j&<dp_k+sum_k^2-2\times sum_i\times sum_k\end{align}$$

也就是

$$\begin{align} 2 \times sum_i \times ( sum_j - sum_k ) &> dp_j + \operatorname { sum }_j ^ { 2 } - dp_k - \operatorname { sum }_k ^ { 2 }\2\times sum_i&>{(dp_j+sum_j^2)-(dp_k-sum_k^2)\over sum_j - sum_k}\end{align}$$

右面这个式子好像一个斜率的形式 (

于是, 我们令 $f_i=dp_i+sum_i^2$, 上式就变成了

$$\begin{align} 2\times sum_i&>{(dp_j+sum_j^2)-(dp_k-sum_k^2)\over sum_j - sum_k}\2\times sum_i&>{f_j-f_k\over sum_j - sum_k}\end{align}$$

然后就直接单调队列维护下凸壳就好了
